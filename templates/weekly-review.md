# Weekly Review Template

## 형식

```
[기간] 주간 회고

📊 이번 주 성과
- [완료한 주요 작업 1]
- [완료한 주요 작업 2]
- [완료한 주요 작업 3]

🎯 주요 결정사항
- [결정 1]: [이유]
- [결정 2]: [이유]

📚 배운 것
- [기술/인사이트 1]
- [기술/인사이트 2]

🔄 개선점
- [잘 안 된 것 + 개선 방향]

📅 다음 주 계획
- [ ] [계획 1]
- [ ] [계획 2]
- [ ] [계획 3]

🏷️ 이번 주 키워드: [키워드1], [키워드2], [키워드3]
```

## 규칙
- ~/claude-telegram/memory/daily/ 파일 기반으로 작성
- 성과는 구체적 결과물 중심
- 결정사항은 "왜"를 포함
- 다음 주 계획은 실행 가능한 수준으로
- 전체 길이: 최대 2000자
- 한국어로 작성
