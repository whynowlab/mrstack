# 스레드 본문 (500자 이내)

Claude Code를 텔레그램 집사로 만들었습니다.

맥북을 덮어놔도, 자고 있어도 24시간 동작합니다.

5분마다 맥북 상태를 스캔하고
→ 코딩 3시간 넘으면 "쉬세요"
→ 배터리 부족하면 "저장하세요"
→ 30분째 커밋 없으면 "막혔어요?"
→ 밤 10시 넘으면 "오늘 그만 하시죠"
→ 딥워크 중이면 알림 자동 차단

매일 밤 내 생산성을 분석해서 코칭 리포트를 보내줍니다.
