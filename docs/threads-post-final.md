# 스레드 본문 (500자 이내)

Claude Code로 24시간 AI 집사를 만들었습니다.
이름은 Mr.Stack.

텔레그램으로 지시하면 코드 작성부터 디버깅까지 해주고,
맥북을 덮어놔도 백그라운드에서 계속 동작합니다.

5분마다 맥북 상태를 스캔해서
→ 코딩 3시간 넘으면 "쉬세요"
→ 30분째 커밋 없으면 "막혔어요?"
→ 밤 10시 넘으면 "오늘 그만하시죠"
→ 딥워크 중이면 알림 자동 차단

대화 내용을 기억하고, 내 작업 패턴을 학습합니다.
일주일 쓰면 내 생산적인 시간대, 자주 하는 요청 유형을 다 파악합니다.
매일 밤 생산성 코칭 리포트도 보내줍니다.

100% 로컬. 서버 없음. API 키 노출 없음.
오픈소스로 공개합니다.
