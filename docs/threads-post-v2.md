이름은 Mr.Stack.

맥북을 덮어놔도, 외출 중이어도, 자고 있어도 동작합니다.

매일 아침 8시에 오늘 일정과 할 일 브리핑을 보내주고,

작업 패턴, 생산적인 시간대, 자주 하는 요청 유형까지 학습합니다.

터미널 안에서만 동작하는 AI가 아닙니다.
24시간 함께 성장하는 AI 파트너입니다.

## 댓글 1 — 전체 기능

Mr.Stack이 할 수 있는 것들입니다.

텔레그램 비서:
• 자연어로 코드 작성/수정/디버깅 지시
• 파일, 사진, 음성 메시지 전부 처리
• 음성으로 지시하면 음성으로 답변 가능 (Whisper + TTS)
• 복사한 내용 자동 분석 (에러 → 원인 분석, URL → 요약, 코드 → 리뷰)

상시 대기 시스템:
• macOS 데몬으로 동작 — 맥북 덮어도, 재부팅해도 자동 실행
• 5분마다 시스템 상태 스캔 (활성 앱, 배터리, CPU, git, 터미널)
• 7가지 상황 자동 감지 + 선제적 알림
• 딥워크 모드 — 집중 중이면 중요한 것만 통과시킴

스케줄 작업 (9개):
• 아침 브리핑 (일정 + 할 일 + AI 뉴스)
• 저녁 요약 + 코칭 리포트
• 3시간마다 대화 분석 → 메모리 자동 업데이트
• 2시간마다 GitHub 알림 확인
• 캘린더 연동 → 미팅 전 리마인더

외부 연동:
• Google Calendar (일정 조회/생성)
• Notion (업무 로그 자동 기록)
• GitHub (PR, 이슈, 알림)
• Playwright (웹 자동화)
• AppleScript (macOS 제어)

메모리 시스템:
• 대화 내용을 기억하고 다음 세션에서 활용
• 사람, 프로젝트, 결정사항, 선호도 자동 분류 저장
• 대화할수록 맥락 이해도가 높아짐

이 모든 게 텔레그램 하나로 동작합니다.

## 댓글 2 — OpenClaw 등 타 솔루션과의 차이

"Claude 텔레그램 봇은 이미 있잖아요?"

맞습니다. 하지만 대부분의 솔루션에는 구조적 한계가 있습니다.

1. 보안
기존 솔루션들은 API 키를 외부 서버에 올리거나, 클라우드를 경유합니다.
Mr.Stack은 내 맥북에서만 동작합니다.
데이터 전송 없음. 서버 없음. 모든 데이터는 로컬 파일(SQLite, JSONL)에만 저장됩니다.

2. 상시 동작
터미널을 열어야만 동작하는 도구가 아닙니다.
macOS LaunchAgent로 등록되어 있어서:
• 맥북 부팅 시 자동 시작
• 크래시 시 자동 재시작
• 맥북을 덮어놔도 백그라운드에서 계속 동작
• 외출 중에도 텔레그램으로 지시 가능

3. 수동 → 능동
대부분의 AI 도구는 "물어봐야 답하는" 구조입니다.
Mr.Stack은 물어보지 않아도 먼저 말합니다.
• 코딩 3시간 → "쉬세요"
• 에러 감지 → "이거 고칠까요?"
• 컨텍스트 전환 과다 → "집중 블록 잡으세요"
• 아침 → 브리핑 / 밤 → 코칭

4. 학습
매 대화가 기록되고 분석됩니다.
생산적인 시간대, 요청 유형 분포, 컨텍스트 전환 패턴...
일주일 쓰면 "나보다 내 업무 스타일을 더 잘 아는" 비서가 됩니다.

## 댓글 3 — 코칭 리포트 예시

매일 밤 9시에 이런 리포트가 텔레그램으로 옵니다.

[Daily Coach] 2026-02-28

생산성: 8/10 (어제 대비 +1)

잘한 점:
• 오전 10-12시 집중 블록 유지 → 커밋 4개
• 스펙을 구체적으로 전달 → 재작업 0회

개선 포인트:
• 14-16시 컨텍스트 전환 7회 — 하루 중 가장 비효율 구간
  → 내일은 이 시간에 단일 작업 블록 추천
• 디버깅 비율 40% — 테스트 먼저 작성 시 절반으로 줄일 수 있음

이번 주 트렌드:
• 평균 지시 명확도: 2.1 → 2.6 (향상 중)
• 가장 생산적 시간대: 10-12시 (일관적)

아첨 없이 직설적으로.
매일 1%씩 나아지는 걸 데이터로 보여줍니다.
