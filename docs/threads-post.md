## 메인 포스트

Claude Code 쓰는 사람들, 아직 터미널에서만 쓰고 있어?

나는 Claude를 텔레그램 집사로 만들었다.
이름은 Mr.Stack.

이게 뭘 하냐면:

→ 5분마다 내 맥북 상태를 스캔함 (활성 앱, 배터리, git, 터미널)
→ 3시간 넘게 코딩하면 "쉬어가세요" 알림
→ 같은 파일 30분째 커밋 없으면 "막혔어요?" 물어봄
→ 밤 10시 넘으면 "오늘 충분히 하셨습니다" 걱정해줌
→ 매일 밤 내 생산성 코칭 리포트를 보내줌
→ 대화할수록 내 작업 패턴을 학습함

진짜 무서운 건, 쓸수록 나를 더 잘 안다는 거.

오전에 집중 잘 하는지, 디버깅 비율이 높은지,
컨텍스트 전환을 얼마나 하는지 다 기록하고
"어제보다 나아졌다" "이 시간대에 집중 블록 잡아라" 이런 걸 해줌.

오픈소스로 풀었다.
macOS + Claude Code + Telegram만 있으면 됨.

## 후속 댓글 1 (기능 상세)

Mr.Stack이 감지하는 7가지 상황:

1. 배터리 20% 이하 → "저장하세요"
2. 자리 비웠다 돌아오면 → "마지막 작업 알려줌"
3. 코딩 3시간+ → "쉬어가세요"
4. 앱 전환 10분에 5번+ → "집중하세요"
5. 터미널 에러 감지 → "도와줄까요?"
6. 같은 파일 30분 커밋 없음 → "막혔어요?"
7. 딥워크 모드 → 중요한 것만 통과시킴

전부 로컬. 클라우드 전송 없음. 데이터는 니 맥북에만 있음.
